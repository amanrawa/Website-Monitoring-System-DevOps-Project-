# 🚀 Website Monitoring System (DevOps Project)

## 📌 Overview
A lightweight DevOps monitoring system built using shell scripting that continuously checks website availability, logs performance, and triggers alerts.

---

## ⚙️ Features
- Monitor multiple websites
- Track HTTP status codes (200, 404, 500)
- Measure response time
- Log uptime/downtime history
- Webhook-based alert system
- Cron job automation
- Dockerized deployment
- Basic analytics dashboard using Python

---

## 🛠 Tech Stack
- Bash Scripting
- Docker
- Linux Cron Jobs
- Python (Pandas, Matplotlib)
- AWS EC2
